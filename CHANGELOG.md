# Changelog — CloudevAI

All notable changes to this project are documented here.

---

## [V1.0.0] — April 2026 — Beta Launch

### Added
- Multi-cloud AI assistant (AWS, Azure, GCP)
- Architecture diagram generation using Mermaid.js
- CI/CD pipeline sequence diagrams
- Terraform and IaC code generation
- Copy button on every CLI command block
- Light and dark mode toggle
- Chat history sidebar (localStorage)
- About section modal with product info
- Security credential exposure alert
- Netlify Serverless Function for API key protection
- Mobile responsive design
- Geist font system

### Security
- API key moved from frontend to Netlify environment variable
- Serverless proxy function added
- Credential detection in system prompt

### Known Issues
- Mermaid sequence diagrams sometimes fail to render
- Diagram fallback shows friendly error message
